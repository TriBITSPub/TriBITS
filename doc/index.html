TriBITS Documentation Index

The TriBITS repository contains the following documentation files:

  • TriBITS Overview Slides (PDF): This is short overivew presentation of TriBITS.
  • TriBITS Overview Poster (PDF): This is a 4 x 4 foot poster providing a high level overview of TriBITS (contains the same material as the above presentation).
  • TriBITS Developers Guide (PDF): This document describes how a TriBITS project is layed out and how to use the features of TriBITS to add new packages, libraries, executables and tests. It also describes how to set up automated pre-push and post-push continuous integation (CI) testing.
  • TriBITS Project Build Quick Reference (PDF): This document described how to configure, build, test, and install piece of software that uses TriBITS as its build, test, and deployment system. This particular document is a generic form of a document that every concrete TriBITS project should generate for their project.
